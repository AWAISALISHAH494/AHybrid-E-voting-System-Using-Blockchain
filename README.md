## Project Structure
- `app.py` - Main Flask application
- `config.py` - Configuration settings
- `models/` - Database models for users, candidates, elections, and votes
- `routes/` - API endpoints for authentication, voting, admin, and blockchain operations
- `services/` - Business logic for encryption, voting, and results
- `blockchain/` - Smart contracts and blockchain server for election results
- `templates/` - HTML templates for UI
- `static/` - CSS and static assets
