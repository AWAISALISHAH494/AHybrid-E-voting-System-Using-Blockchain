{{ icons.icon_info() }} How Verification Works

1. When an election is finalized, all votes are counted and the results are hashed using SHA-256.

2. This hash is stored permanently on the Ethereum blockchain via a smart contract.

3. To verify, we recalculate the hash from current database results and compare it with the blockchain record.

4. If the hashes match, the results are proven to be tamper-free. If they don't match, data may have been altered after finalization.
